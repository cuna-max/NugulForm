상세 설명:

NugulForm - Google Forms 자동 채우기 도구

반복 입력은 이제 그만! NugulForm은 Google Forms 작성 시 매번 입력해야 하는 개인 정보를 자동으로 기입해주는 Chrome 확장 프로그램입니다.

주요 기능:

- 자동 정보 기입: Telegram Handle, Twitter Handle, Google 계정, 지갑 주소(EVM, Sol 등), YouTube, 전화번호 등을 한 번 저장하고 Google Forms에서 자동으로 기입
- 스마트 필드 매칭: 다양한 형태의 폼 라벨을 자동으로 인식하여 정확한 필드에 정보를 기입
- 자동 선택지 선택: "예", "네", "Yes" 등 긍정 응답을 자동으로 선택
- 클립보드 복사: 저장된 정보를 빠르게 복사하여 수동 입력 가능
- 사용자 정의 필드: 자주 사용하는 커스텀 정보를 추가하여 자동 기입 가능
- 안전한 저장: 모든 정보는 브라우저에 로컬 저장되며 외부 서버로 전송되지 않습니다

사용 방법:

1. 확장 프로그램 설치 후 옵션 페이지에서 개인 정보 입력
2. Google Forms 페이지에 접속
3. 우측 하단의 자동 채우기 버튼 클릭
4. 저장된 정보가 자동으로 채워집니다

지원 사이트: https://docs.google.com/forms/* (Google Forms 전용)

개인정보는 브라우저에만 저장되며 외부로 전송되지 않습니다. 안심하고 사용하세요!
